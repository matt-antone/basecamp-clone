## 1. SECURITY

### CRITICAL
- **Admin import endpoints** (`app/admin/imports/basecamp2/route.ts:7`, `[jobId]/route.ts:7`, `[jobId]/retry-failed/route.ts`) ‚Äî only `requireUser` guard. Add real admin role check backed by DB.
- **Cookie `secure` flag tied to `NODE_ENV`** (`lib/server-auth.ts:13-17,104-105`) ‚Äî previews transmit access/refresh tokens over plaintext. Set `secure: true` unconditionally outside dev.

### HIGH
- **Path-traversal via `targetPath`** (`app/projects/[id]/files/upload-complete/route.ts:14-38`) ‚Äî reconstruct path server-side from `project.storage_dir` or assert `startsWith` + reject `..`, `//`, backslashes, nulls.
- **Download-link open ACL** (`app/projects/[id]/files/[fileId]/download-link/route.ts:19-23`) ‚Äî enforce project membership.
- **No authorization model** ‚Äî no `isAdmin` / `isMember(projectId,userId)`. Every workspace user = effective admin. Introduce roles + per-resource guard.
- **`site-settings` PATCH** (`app/site-settings/route.ts:41-48`) admin-only.
- **Client archive/restore** (`app/clients/[id]/archive/route.ts`, `restore/route.ts`) destructive, admin-only.

### MEDIUM
- **`sanitize-html` config** (`lib/markdown.ts:9-15`) allows `input[type|checked|disabled]`, keeps defaults for `a`/`img`. Tighten: drop `input`, force `rel="noopener noreferrer"`, `https:` only on `img[src]`, strip `target`.
- **Error leakage** ‚Äî `unauthorized(error.message)` in admin imports exposes internal strings. Return generic, log server-side.
- **No CSRF / Origin checks**. Cookies are not `sameSite: "strict"`. Cross-site POSTs with cookies land. Add Origin allowlist or strict cookies for state mutations.
- **`scripts/mint-mcp-jwt.mjs`** no min-secret-length, no max `expiresIn`. Require `secret.length >= 32`.
- **No rate limiting** anywhere. Add per-IP/per-user limiter on callback, comments, uploads.

### LOW / INFO
- No `middleware.ts` ‚Äî per-handler gating is easy to forget on new routes.
- No Supabase RLS (zero `ENABLE ROW LEVEL SECURITY` hits). Add as defense-in-depth.
- `bcryptjs` listed but unused in repo ‚Äî remove if truly dead.
- `components/discussions/markdown-html.tsx:10` uses React's raw-HTML injection prop ‚Äî sanitize at boundary, not just upstream.

### Posture
Auth strong (Supabase PKCE + workspace allowlist + httpOnly). Authz binary: workspace = admin. Zero defense-in-depth (no RLS, CSRF, rate limit, strict cookies). Main exploit surfaces: missing role gates and client-supplied paths.

## 3. PERFORMANCE

### Deploy target ambiguity
`.vercel/` exists AND `netlify.toml` exists. No `vercel.json`. Delete whichever isn't canonical.

### WINS
1. **Every route is client-rendered SPA shell.** All `page.tsx` = `"use client"` + `createClientResource` ‚Üí `loadProjectBootstrap` ‚Üí 6x parallel `authedJsonFetch`. Critical path = HTML ‚Üí JS ‚Üí hydrate ‚Üí auth ‚Üí 6 fetches ‚Üí render. Convert to RSC; call `lib/repositories.ts` directly, pass `initial` to small client wrapper. Estimated LCP ‚àí1.5 to ‚àí3s. Files: `app/page.tsx`, `app/[id]/page.tsx:964-985`, `app/[id]/[discussion]/page.tsx`, `app/archive/page.tsx`, `app/flow/page.tsx`, `app/billing/page.tsx`, `app/settings/page.tsx`.
2. **`@mdxeditor/editor` CSS imported globally** (`app/styles.css:1`). Move into `components/markdown-editor.tsx` so it ships with the editor chunk only.
3. **`next/image` unused.** Raw `<img>` in header logo (likely LCP), avatars, thumbnails. `app/header.tsx:224`, `app/settings/_settings-page-content.tsx:544`, `app/[id]/page.tsx:559,620,741`, `components/file-thumbnail-preview.tsx:184`. Migrate + add `priority` on logo + configure `remotePatterns`.
4. **Site settings client-fetched per page** (`app/header.tsx:89`, `app/settings/page.tsx:140` with `cache: "no-store"`). Fetch in `app/layout.tsx`, pass via context. Wrap hot read with `'use cache'` + `cacheTag('site-settings')`, `revalidateTag` on PUT.

### MEDIUM
- `force-dynamic` on `app/feeds/latest/route.ts:4` defeats upstream `revalidate: 86400`. Remove.
- Zero `<Suspense>` boundaries in `app/`. After RSC migration, stream sub-sections.
- No cache tags / `unstable_cache` / `'use cache'` in `lib/repositories.ts`. Add for hot reads.
- `pg` pool `max: 5` (`lib/db.ts:12`) ‚Äî consider Supabase pgBouncer transaction pooler.
- `renderMarkdown` (marked + sanitize-html) shipped to client in `app/[id]/page.tsx`. Render server-side, send `body_html` only.

### LOW
- `beforeInteractive` inline theme script in `app/layout.tsx:31` ‚Äî minor cost.
- Sequential `await query()` loops in `lib/imports/basecamp2-import.ts:104,139,180,226`. Batch or `Promise.all` chunks.

### Strategic
- Rendering model is root cause of ~80% of perf upside. Full CSR app in App Router clothing.
- Priority: (1) `app/[id]/page.tsx` + `[discussion]/page.tsx` ‚Üí RSC, (2) tag-based caching, (3) `next/image`.
